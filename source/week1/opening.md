# 开光大会总结
## 关于课程
- 主题是 Python, 不是其它
  - `集中主要精力在 Python 上`
- 正式课程内容是小任务

## 关于翻转课堂
- 自学与授课的思路并不一定相同
  - `给予导师们充分的信任(迷信)`
- 具体写书时的重点: 写给六个月前的自己
  - `自己/六个月前的自己有什么特点?`
     - `对于自己的特点来写书`
     - `不必过分细节/考虑通用性,但求自己能理解`
     - `另一方面,近期时间确实太有限,不得不控制得精细一些`
